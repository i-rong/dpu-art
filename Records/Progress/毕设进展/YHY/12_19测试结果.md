## 前言

实验记录：2023年12月19日，在 `dpa_kernel_launch` 的基础上，测试 Host 与 DPA 通信的平均时延，记录循环不同次数的平均时延。

## 实验结果

| 循环次数 | 平均时延(ms) |
| -------- | ------------ |
| 1        | 1.539000     |
| 10       | 0.51960      |
| 100      | 0.439920     |
| 1000     | 0.400739     |
| 10000    | 0.398930     |
